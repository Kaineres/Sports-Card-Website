# GemRate — Universal Pop Report API

## What GemRate Is

GemRate is "the trading card industry's first and only Universal Pop Report" — a data platform aggregating population report data across all four major grading services (PSA, Beckett/BGS, SGC, CGC) in a single unified dataset, updated daily.

Website: gemrate.com

## Why This Matters for SlabMetrics

PSA's official Public API has NO population report endpoint — cert lookup only. GemRate is the primary viable path to programmatic pop report data for Wedge Signal #4 (pop report changes as a Buy/Hold/Sell signal).

Additionally, GemRate's universal card IDs map cards across graders and marketplaces, which is a significant data infrastructure asset — it solves the "same card, different cert numbers across PSA/BGS/SGC" identity problem.

## What GemRate Provides

- **Universal Pop Report**: PSA, Beckett (BGS), SGC, and CGC populations returned in a single API call
- **Updated daily**: population counts reflect recent grading submissions
- **Universal IDs**: stable card identifiers that map across graders and marketplaces — critical for cross-grader price comparison
- **Daily grading trends**: velocity data showing how submission rates are changing

## Access Model

**Partner API** — not self-serve. Requires:
- Direct outreach via gemrate.com contact form
- Demo booking with their team to walk through use case and pricing
- Pricing is not public

GemRate is a B2B data product, not a developer-tier API. Access is relationship-based.

## SlabMetrics Use Case

GemRate is the **only known API** that provides programmatic, multi-grader population report data in one call. For Wedge Signal #4:
- Poll GemRate daily for population changes on cards in the SlabMetrics catalog
- A significant population increase (many new PSA 10s submitted) = supply increasing = potential sell signal or price pressure
- A rare card maintaining low PSA 10 population = scarcity holding = potential buy/hold signal

GemRate's universal IDs also solve a hard data modeling problem: mapping the same card (e.g., "2023 Patrick Mahomes Prizm #1") across PSA cert numbers, BGS cert numbers, and eBay listing identifiers.
