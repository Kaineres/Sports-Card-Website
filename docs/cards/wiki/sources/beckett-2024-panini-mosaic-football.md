## TL;DR

- 400-card chrome set; hobby box delivers 2 autographs, 20 inserts, 15 Mosaic Prizm parallels, 5 Silver Prizm parallels across 15-card packs × 10 packs. `[Certain]`
- Base set has 4 named subsets: National Pride (#251–270), NFL Debut (#271–290), Hall of Fame (#291–300), Rookies (#301–400). `[Certain]`
- 38 base parallel variants — unprinted rainbow down to 1/1 (Black Mosaic, Nebula Choice, No Huddle Black). `[Certain]`
- Autograph sets: Rookie Autographs (100 cards), In Focus Signatures, Showtime Signatures, Scripts — each with numbered parallel stacks. `[Certain]`
- Short-print inserts: Money, Micro Mosaic, Stained Glass. `[Certain]`

- Set size: 400 cards. `[Certain]`
