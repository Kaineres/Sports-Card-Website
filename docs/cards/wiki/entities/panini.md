# Panini

Panini America is a major US trading card manufacturer holding NFL and NBA licenses. Sets are characterized by chrome construction and extensive parallel rainbow structures with many numbered and 1/1 variants.

## TL;DR

Panini produces NFL, NBA, and other licensed card products. Their releases feature "parallel rainbows" — the same card reprinted across many colored or patterned variants at successively lower print runs, creating a rarity hierarchy within a single card number.

## Key brands

- **Mosaic** — chrome set with the tile-pattern (mosaic) parallels alongside standard Silver Prizm parallels. `[Certain]` — [[sources/beckett-2024-panini-mosaic-football]]
- **Prizm** — flagship chrome brand (referenced in product descriptions but not yet a source-ingested page). `[Likely]`

## Parallel structure

Panini sets use a "parallel rainbow" model: the same card exists in many variants with decreasing print runs. Numbered parallels carry their print run stamped on the card (/199, /99, /49, /25, /10, /8, etc.); 1/1 variants are unique one-of-ones. A single Mosaic Football base card has 38 named parallel variants. `[Certain]` — [[sources/beckett-2024-panini-mosaic-football]]
