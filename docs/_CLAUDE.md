## Section 0 — AI-First Vault Rule

This vault is designed for **future-Claude** to read and reason over. Every note Claude writes must follow these rules:

1. **Self-contained context** — each note explains itself without relying on backlinks alone
2. **"For future Claude" preamble** — 2-3 sentence summary at the top so Claude can judge relevance in 10 seconds
3. **Rich frontmatter** — `type`, `tags`, `updated`, `certainty`, `sources` on every page
4. **Recency markers per claim** — e.g. "PSA has 14 service tiers (as of 2026-05, psacard.com)" so future-Claude knows what to verify
5. **Sources preserved verbatim** — every external claim cites `[[sources/<slug>]]`
6. **Cross-links mandatory** — every entity/concept/topic referenced uses `[[wikilinks]]`
7. **Certainty levels** — `[Certain]` / `[Likely]` / `[Guessing]` on every factual claim
